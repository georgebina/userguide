  4. Oxygen XML Author Takes Several Minutes to Start

Oxygen XML Author Takes Several Minutes to StartEdit on GitHub

Problem: Some anti-virus software can cause Java applications, such as Oxygen XML Author, to start very slowly due to scanning of compressed archives (such as the JAR libraries that all Java applications use).

Solution: The solution is to add the Oxygen XML Author folder to the list of exceptions in the anti-virus software settings.
