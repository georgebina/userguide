DITA 1.3 Support
Edit on GitHub

Starting with version 17.1, Oxygen XML Author includes support for some DITA 1.3 features.

To enable DITA 1.3 support in Oxygen XML Author and use the DITA Open Toolkit 2.x for publishing, open the Preferences dialog box (Options > Preferences), go to DITA, and select the Built-in DITA OT 2.x radio button.

The Oxygen XML Author publication of the full DITA 1.3 specifications can be found at http://www.oxygenxml.com/dita/1.3/specs/index.html#introduction/dita-release-overview.html.
